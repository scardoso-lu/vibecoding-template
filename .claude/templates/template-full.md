# Full Feature Template Index

This file is intentionally small. Do not put category-specific template bodies here.

In Plan Mode, read `.claude/templates/template-routing.md` first, then load only the category templates required by the current slice.

Normal non-minimal feature memory still produces only:

```text
feature-memory/<slice>/
  slice.md
  rules.md
  e2e-coverage.json  # user-facing slices only; maps initial prompt stories to Playwright tests
  qa-evidence.json   # generated by scripts/validate/gate.py after deterministic checks run
```

The category templates provide sections to compose into `slice.md` and `rules.md`; they are not separate output files.
